# Grid Tycoon

**Grid Tycoon** is a fast-paced arcade city builder. Place buildings to serve waves of customers, optimize your layout with adjacency bonuses, and maximize revenue before time runs out.

## How to Play

*   **Goal:** Earn as much money as possible in 3 minutes.
*   **Controls:**
    *   **Click/Tap:** Place selected building.
    *   **Right-Click:** Sell building.
    *   **1-4 Keys:** Select building type.
    *   **Palette:** Tap buttons at bottom to select building.

## Buildings

1.  **House ($50):** Spawns customers periodically. Essential for demand.
2.  **Shop ($100):** Serves customers to earn money. Has limited capacity.
3.  **Park ($75):** Increases revenue of adjacent shops (+25%).
4.  **Road ($10):** (Cosmetic/Planning for MVP) Cheap placeholder.

## Tips

*   Customers will walk to the nearest shop. Ensure they have a clear path!
*   If a shop is full (queue full), customers will wander away. Build more shops!

# Terraform

A fast-paced arcade farming and merging game.

## Controls

- **Tap Empty Tile**: Plant a seed (Tier 1).
- **Tap Crop**: Merge with adjacent identical crops to upgrade tier.
- **Double Tap Crop**: Harvest current crop (and chain) for points.

## Goal

- Maximize your score in 90 seconds.
- Chain merges to reach high tiers (Tier 6+).
- Harvest large clusters quickly for combo multipliers.

## Scoring

- **Harvest Value**: `Base * 2^(Tier-1)`
- **Chain Bonus**: Harvest more connected tiles for multiplier.
- **Combo**: Rapidly harvest multiple groups for growing multiplier.

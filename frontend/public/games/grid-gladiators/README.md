# Grid Gladiators

Tactical arena duels on a tile grid.

## Rules
- **Objective**: Eliminate the opponent or have higher score when time runs out.
- **Movement**: WASD (P1) or Arrows (P2).
- **Abilities**: Each gladiator has 2 unique abilities.
- **Tiles**:
  - **Cover**: Blocks shots.
  - **Trap**: Damages you.
  - **Bounce**: Pushes you fast.
  - **Heal**: Regenerates HP.

## Controls
- **Player 1**: WASD to move, Space for Ability 1, Shift for Ability 2. Mouse aiming supported.
- **Player 2**: Arrow keys to move, Enter for Ability 1, Ctrl for Ability 2.

## Gladiators
1. **Bruiser**: High HP, Dash Smash, Shield.
2. **Sharpshooter**: Low HP, Long Range Snipe, Stun Grenade.
3. **Trickster**: Medium HP, Blink, Trap Mine.

## URL Parameters
- `?seed=123`: Set PRNG seed.
- `?mode=pvp`: Enable local 2-player mode. (Default is `pve`).

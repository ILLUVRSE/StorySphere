# Spy Grid

**Spy Grid** is a compact, tactical stealth puzzle game.

## Gameplay
- **Goal**: Sneak past guards, collect objectives, and reach the exit.
- **Controls**:
  - **Desktop**: Arrow keys or WASD to move. Space to wait/interact.
  - **Mobile**: Tap tiles to move.
- **Rules**:
  - Guards have a vision cone. If you step into it, you are detected (Game Over in Hard Mode).
  - Guards patrol fixed routes.
  - Tiles provide cover or utility (vents, noise).

## Scoring
- Base score for objectives.
- Bonus for stealth (undetected).
- Bonus for efficiency (fewer turns).

# Pixel Puck

Fast-paced 1v1 air-hockey style matches on a tile grid.

## Controls

### Single Player
- **Move:** WASD (or Arrow Keys)
- **Dash/Charge:** Space (or Enter)
- **Mouse/Touch:** Drag to move paddle directly. Tap/Click to Dash.

### Local Multiplayer
- **Player 1:** WASD + Space
- **Player 2:** Arrow Keys + Enter / Shift

## Rules
- Score goals to win.
- Matches last 90 seconds.
- Most goals wins.
- Sudden death if tied at time limit.

## Features
- **Power Tiles:** Speed up or slow down the puck.
- **Bumpers:** Obstacles that bounce the puck.
- **Dash:** Short burst of speed to hit the puck harder.
